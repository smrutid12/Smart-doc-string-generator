# Smart Docstring Generator


A lightweight, AI-powered tool that generates clean and consistent
docstrings using the **Grok (LLaMA-based)** model.

## 🚀 Features

-   Supports **Google**, **NumPy**, and **PEP‑257** docstring styles.
-   Works with multiple languages: **Python, JavaScript, TypeScript,
    Java, C, C++**.
-   Accepts **direct code input** or **file upload**.
-   Produces accurate, readable, and structured documentation.
-   Improves code clarity and maintainability with minimal effort.

## 🛠 How It Works

1.  Choose your preferred docstring style.
2.  Select the programming language.
3.  Paste code or upload a file.
4.  Let the AI generate optimized docstrings instantly.

## 📦 Tech Stack

-   **FastAPI** -- backend API\
-   **Grok (LLaMA-based model)** -- docstring generation\
-   **Frontend** -- modern nextJS UI for code input and file upload

## 📄 Output Example

The tool automatically analyzes functions, classes, and modules to
produce high‑quality docstrings tailored to your selected style.
